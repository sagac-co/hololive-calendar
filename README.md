### 📱 如何訂閱

**Apple 行事曆（Mac / iPhone / iPad）**
1. 點擊上方「點此訂閱」連結
2. 系統會詢問是否用行事曆開啟，點「訂閱」即完成 ✅
3. 之後會自動更新

**Google 行事曆（Android / 網頁）**
1. 點擊上方「點此訂閱」連結，複製網址
2. 開啟 [calendar.google.com](https://calendar.google.com)
3. 左側「其他日曆」旁的 **＋** → **透過網址新增**
4. 貼上網址 → 點「新增日曆」✅

> ⚠️ Google 行事曆的同步頻率約 8～24 小時更新一次，這是 Google 的限制。
